Introducere

CSS sau Cascading Style Sheets este un standard folosit pentru a specifica prezentarea și stilul unui document scris într-un limbaj markup, cum ar fi HTML sau XML. CSS este o tehnologie de bază a World Wide Web, alături de HTML și JavaScript. CSS este conceput pentru a permite separarea conținutului și a prezentării, inclusiv aspectul, culorile și fonturile. Această separare poate îmbunătăți accesibilitatea conținutului, oferă mai multă flexibilitate și control în specificarea caracteristicilor de prezentare. Numele în cascadă provine din schema de prioritate specificată pentru a determina ce declarație se aplică dacă mai multe declarații ale unei proprietăți se potrivesc cu un anumit element. Această schemă de prioritate în cascadă este previzibilă. Specificațiile CSS sunt menținute de World Wide Web Consortium (W3C). Pe lângă HTML, alte limbaje de marcare acceptă utilizarea CSS, inclusiv XHTML, XML simplu, SVG și XUL. CSS este folosit și în setul de instrumente widget GTK. 

Sintaxă  

CSS are o sintaxă simplă și folosește un număr de cuvinte cheie în limba engleză pentru a specifica numele diferitelor proprietăți de stil. Mai jos sunt prezente in listă cele mai importante module:

  • Selectori
  • Clase
  • Culori
  • Margini
  • Backgrounds and Borders
  • Imagini
  • Butoane
  • Opacitate
  • Fonturi
  • Linkuri
  • Liste
  • Tabele
  • Efecte pentru text
  • Transformări 2D/3D
  • Animații
  • Prezentare pe mai multe coloane
  • Interfața cu utilizatorul
Am menționat clasele. O clasă CSS este un set reutilizabil de reguli de stil care poate fi aplicat unuia sau mai multor elemente HTML. Servește ca o modalitate de a grupa elemente și de a aplica stiluri consistente. Imaginați-vă că aveți mai multe titluri în conținutul site-ului dvs. pe care doriți să le stilați în același mod. În loc să scrieți aceleași reguli CSS pentru fiecare dintre ele în mod individual, puteți crea o clasă CSS pentru a defini acele stiluri o dată și apoi să o aplicați fiecărui element de titlu. Istoric

HTML nu a fost niciodată destinat să conțină etichete pentru formatarea unei pagini web. HTML a fost creat pentru a descrie conținutul unei pagini web (titluri, paragrafe, liste, tabele etc.) Când etichete precum <font> și atribute de culoare au fost adăugate la specificația HTML 3.2, a început un coșmar pentru dezvoltatorii web. Dezvoltarea de site-uri web mari, în care fonturile și informațiile de culoare au fost adăugate la fiecare pagină, a devenit un proces lung și costisitor. Pentru a rezolva această problemă, World Wide Web Consortium (W3C) a creat CSS. CSS a eliminat formatarea stilului din pagina HTML! Pe 17 decembrie 1996, W3C a publicat primul standard pentru CSS.
